# 糖域卫士 - Vercel部署指南

## 快速部署

### 方法1：使用Vercel CLI（推荐）

```bash
# 1. 安装Vercel CLI
npm install -g vercel

# 2. 登录Vercel
vercel login

# 3. 部署到生产环境
vercel --prod
```

### 方法2：通过GitHub部署

1. 将项目推送到GitHub仓库
2. 访问 https://vercel.com
3. 导入GitHub仓库
4. 自动部署完成

## 访问地址

部署成功后，你的项目将可以通过以下地址访问：
- 主页面：https://你的项目名.vercel.app/tangyu.html
- 登录页面：https://你的项目名.vercel.app/simple_login.html
